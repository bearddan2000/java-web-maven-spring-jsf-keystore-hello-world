# java-web-maven-spring-jsf-keystore-hello-world

## Description
A POC for springboot web app with jsf primefaces support with self-signed ssl.

## Tech stack
- java
- maven
  - springboot
  - jsf
  - bootstrap

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
[Available at](https://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
